# Video Prompting Guide

## Video Prompt Structure

### Core Elements
1. **Subject**: What/who is in the video
2. **Action**: What's happening
3. **Setting**: Where it takes place
4. **Style**: Visual aesthetics
5. **Camera**: Movement and framing

### Example Prompts

**Basic**:
```
A golden retriever running through a sunlit meadow
```

**Detailed**:
```
A golden retriever running joyfully through a sunlit meadow,
slow motion, cinematic lighting, shallow depth of field,
drone tracking shot following the dog, 4K quality,
summer afternoon, wildflowers in the background
```

## Style Keywords

### Cinematic
- `cinematic`, `film grain`, `anamorphic lens`
- `dramatic lighting`, `high contrast`
- `shallow depth of field`, `bokeh`

### Documentary
- `documentary style`, `handheld camera`
- `natural lighting`, `raw footage`

### Commercial
- `professional`, `polished`, `clean`
- `product showcase`, `advertising quality`

### Artistic
- `artistic`, `creative`, `experimental`
- `abstract`, `surreal`, `dreamlike`

## Camera Movements

- `tracking shot` - follows subject
- `dolly in/out` - moves toward/away
- `pan left/right` - rotates horizontally
- `tilt up/down` - rotates vertically
- `crane shot` - sweeping overhead
- `steadicam` - smooth handheld
- `drone aerial` - bird's eye view
- `static shot` - fixed position

## Shot Types

- `extreme close-up` - detail focus
- `close-up` - face/object
- `medium shot` - waist up
- `full shot` - entire subject
- `wide shot` - subject + environment
- `establishing shot` - location context

## Lighting

- `golden hour` - warm sunset/sunrise
- `blue hour` - twilight
- `high key` - bright, minimal shadows
- `low key` - dark, dramatic shadows
- `backlit` - light from behind
- `rim lighting` - edge highlighting
- `natural light` - realistic

## Music Prompting

### Structure
```
[Genre] music for [context], [mood], [instruments], [tempo]
```

### Examples

**Epic**:
```
Cinematic orchestral music for movie trailer,
epic and emotional, full orchestra with choir,
building intensity, 90 BPM
```

**Ambient**:
```
Atmospheric ambient music for documentary,
contemplative and serene, synthesizers and piano,
slow tempo, minimal percussion
```

**Corporate**:
```
Upbeat corporate music for product video,
professional and optimistic, acoustic guitar
and light percussion, 120 BPM
```

## Scene Breakdown Strategy

### For Long-Form Videos

Break concept into scenes with:
1. **Opening hook** - Grab attention (4-8s)
2. **Setup** - Establish context (8-16s)
3. **Development** - Main content (varies)
4. **Climax** - Key moment (8-16s)
5. **Resolution** - Wrap up (4-8s)

### Scene Transition Tips

- Match movement direction
- Consistent lighting tone
- Similar color palette
- Complementary composition

## Negative Prompts

What to avoid:
```
blurry, low quality, distorted faces,
unnatural motion, glitchy, watermark,
text overlay, logo, amateur
```

## Content Policy

Avoid:
- Violence or gore
- Explicit content
- Celebrity likeness
- Copyrighted characters
- Real minors
- Hate symbols
